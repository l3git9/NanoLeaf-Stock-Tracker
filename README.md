# NanoLeaf Stock Tracker
 
Hi All! 
Thank you for checking out the code. It's super simple so it should be easy to digest and get into. I was super surprised by the capabilities of the nanoleaf and just the whole process of doing this.
Ahead are some necessary steps you will need to take in order to use this application as their is a tiny bit of set up and nothing I can do to solve that problem. 


INSTRUCTIONS: 
1. Log into router and look for ip address nanoLeaf. 
2. Use program such as post man in order to get authentication token. Use the following url and make sure your call is a POST: http://'YOUR IP HERE'/api/v1/new 
3. Finally insert your IP and auth token into your config file and you should be all set :).
